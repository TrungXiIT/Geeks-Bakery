# Geeks-Bakery
 Geek's Bakery - My bakery website for The Rookies Program

 ## Tech Stack
 ### ASP .NET Core (Backend - API - Client site (Razor))
 - MVC - Razor Page (Client site)
 - REST API
 - ASP.NET Core 5.0
 - Entity Framework Core 5.0.7
 - SQL Server
 ### ReactJs (Frontend - Admin site)
 - ReactJS (Admin site)
 - ReactStrap
 - CSS
